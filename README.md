# Prism ContentEditable Editor

A lightweight code editor built with HTML, CSS, JavaScript, and Prism.js.

## Features

* Real-time syntax highlighting
* ContentEditable-based editing
* Caret position preservation after highlighting
* Mobile-friendly interface
* Automatic line wrapping
* No build tools required

## Usage

Open `index.html` in any modern browser.

## Technologies

* HTML5
* CSS3
* JavaScript
* Prism.js

## How It Works

The editor uses a `contenteditable` element for text input and Prism.js for syntax highlighting. After each input event, the caret position is saved, the code is highlighted, and the caret is restored to its previous position.

## Limitations

* Designed for small to medium code snippets
* Currently tested with JavaScript highlighting
* Not intended as a full IDE replacement
